One-Click Publish to GitHub
Set this up once. After that, click Publish to GitHub and your changes go live on all devices in 2 minutes.
How to get a GitHub Token (one time only):
1. Go to github.com → click your profile photo (top right) → Settings
2. Scroll to bottom of left sidebar → click Developer settings
3. Click Personal access tokens → Tokens (classic)
4. Click Generate new token (classic)
5. Give it a name like "DPT Admin" → set expiration to No expiration
6. Tick only the repo checkbox
7. Click Generate token → copy the token immediately (you won't see it again)
Stored only in your browser — never sent anywhere except GitHub's API.
